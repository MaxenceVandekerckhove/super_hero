Bonjour et bienvenue dans le fichier de l'application POWERMAP.

Dans ce fichier vous trouverez : 

Un dossier Documentation : Contenant différents documents tels qu'un cahier de test, un PowerPoint, Un document de Coordonées de Tests, ou encore le cahier des charges du projet.

Afin de faire fonctionner le projet sur votre environnement :

Vous aurez besoin d'une base de donnée MySQL local, pour celà vous pouvez utiliser WAMP que vous pouvez installer à cette adresse :
https://www.wampserver.com/

De plus aurez besoin d'éxecuter le projet ainsi que le repositoryTest permettant l'implémentation des données de base ( Incidents ainsi que 3 Héros Tests ).
